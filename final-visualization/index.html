INFO474: Seattle Collision Visualization

Daniel Pham, Kevin Yip, & Jonathan Thomas

Background

The dataset used for this project is called “Collisions,” retrieved from data.gov, a U.S. government website that aims to improve public access to datasets generated by the Executive Branch of the Federal Government. The dataset provides information about collisions that have occurred in the Seattle area from January 2004 to September 2021, collected by the Seattle Police Department.

The dataset is a geographic traffic collision dataset in CSV form. There are a total of 40 attributes and approximately 230,000 rows of data. Examples of notable attributes include location coordinates, weather type, road conditions, number of individuals involved, and severity description. To set up the neighborhood grids, a neighborhood geoJSON was used to construct the map. Additionally, the collisions dataset was modified to include information about which neighborhood the collision occurred in. Neighborhood name and center point were added to the original dataset's CSV file using d3 geocontain for more context.

How to Interact with Visualizations

  • Collision Analytics Card: the card shown on the left shows vital statistics for a selected region. Select either a bubble/dot on the map, and the card will display information relevant to the neighborhood the bubble/dot is located in. Click "Go to Location" button to make the map zoom to the location. To zoom out, click "All of Seattle."
  • Map: the map in the middle shows the areas in Seattle that have had collisions. There are two modes for the graph: neighborhood cases and individual cases. Neighborhood cases is the default view, and is shown with bubbles that vary in size based on the number of collision instances in a neighborhood. Individual cases show dots for every collision, which can be seen by toggling the "Individual Cases" switch below the analytics card. To avoid accidental clicks on neighborhoods while scrolling through the map, switch on "Drag Mode."
  • Filtering: the data can be filtered by each neighborhood using the dropdown, along with "Month Range" and "Date Range" sliders that can be adjusted from either side. To reset the data to view for all of Seattle area, click "Default View."
  • Bar Chart: the bar chart shown below the map shows the collision type versus the severity description for the Seattle area or a particular neighborhood that is selected. To view particular neighborhood cases, click on a neighborhood grid or dot point on the map, or select a neighborhood from the dropdown menu under "Filter by Neighborhood." Hover over each portion of the bar chart to view the exact count of a severity description.
